# CatApp: Discover Your Feline Knowledge

**CatApp** is a feature-rich Android application developed for the **"Mobile Apps Development"** university course. The app offers a experience in an educational journey through various cat breed.

---

## Key Features

### User Management
* **Onboarding**: Local account creation upon the first launch.
* **Profile Management**: View and edit profile details.

### Cat Breeds Explorer
* **Comprehensive List**: Browse all cat breeds with high-quality previews and a search filter.
* **Deep Insights**: Detailed information for every breed, including temperament and history.
* **Media Gallery**: Integrated photo gallery with a dedicated full-screen photo viewer.

## Technical Stack & Architecture

This project was built using the latest Android development standards to ensure scalability and performance:

* **Architecture**: **Model-View-Intent (MVI)** for predictable state management and reactive UI.
* **Asynchronous Logic**: **Kotlin Coroutines** for efficient background processing.
* **UI Framework**: **Jetpack Compose** for a modern, declarative user interface.
* **Local Database**: **Jetpack Room** for persistent data storage.
* **Preferences**: **Jetpack DataStore** for managing local user profiles and settings.
* **Dependency Injection**: **Hilt (Dagger)** for clean, modular dependency management.
